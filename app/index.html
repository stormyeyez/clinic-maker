Beyond payments — Hermes can also provision what it needs

Stripe's newer Stripe Skills for Hermes go beyond payments — they let an agent provision its own infrastructure (a database, hosting, messaging) directly from the terminal, with Stripe acting as the trusted identity layer between the agent and the provider. We have this installed and ready:

stripe projects catalog — 78 real services across 19 categories neon/postgres · free tier database vercel/project · free tier hosting twilio/sms · patient text follow-ups

Honest limitation: actually provisioning a live resource this way requires the Stripe account to complete identity verification (KYC) — a real, multi-step process on Stripe's side, not something we could responsibly rush through for this demo. So this capability is real and installed, but not executed live here.
